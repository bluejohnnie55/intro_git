Hello world

.gitファイルを作成する（'.'は隠しファイル）
$ git init

基本的なワークフロー
１．ファイルの変更をステージングエリアへ追加する
$ git add (file name)
$ git add . ('.' means 'all files')

２．ローカルリポジトリにコミットする
$ git commit -v　（'-v'をつけると変更箇所が確認できる）
$ i (insert: 'i')
$ type comment
$ type escape
$ type ':wq' and Enter to finish

３．リモートリポジトリにプッシュする
$ git remote add origin https://github.com/bluejohnnie55/intro_git.git
$ git push -u origin master

４．ログの確認をする
$ git log

    色々なオプション

    # 一行で表示する
    $ git log --oneline

    # ファイルの指定をする
    $ git log -p (file name)

    # 表示するログ数を指定する
    $ git log -n (int)

５．ステータスを確認する
$ git status

６．ファイルの削除をする
$ git rm (file name)

７．ファイルの変更差分を表示する
    # difference
    $ git diff

    # ステージとコミットの差分
    $ git diff HEAD
